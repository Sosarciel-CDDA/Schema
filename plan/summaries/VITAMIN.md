# VITAMIN.md 总结

## 文档概述

维生素系统定义了角色体内的各种营养素、毒素、药物和计数器，包括缺乏症和过量症的效果。

---

## 大类列表

| 类型 | 说明 |
|------|------|
| `vitamin` | 维生素，定义营养素/毒素/药物/计数器的属性、范围、疾病阈值等 |

---

## vit_type 枚举

| 值 | 说明 |
|------|------|
| `vitamin` | 营养维生素(简化营养模式下忽略) |
| `toxin` | 毒素化学物质(目前无效果) |
| `counter` | 计数器(非毒素/维生素/药物) |
| `drug` | 药物(需30分钟生效) |

---

## 关键字段

| 字段 | 说明 |
|------|------|
| `id` | 唯一ID |
| `type` | 必须为 `vitamin` |
| `vit_type` | 维生素类型 |
| `name` | 显示名称 |
| `deficiency` | 缺乏症效果ID |
| `excess` | 过量症效果ID |
| `min` | 最小值 |
| `max` | 最大值 |
| `weight_per_unit` | 单位重量 |
| `rate` | 自然衰减速率 |
| `flags` | 标志数组 |
| `disease` | 缺乏症阈值数组 |
| `disease_excess` | 过量症阈值数组 |
| `decays_into` | 代谢产物数组 |

---

## flags

| 标志 | 说明 |
|------|------|
| `NO_DISPLAY` | 检查食物时不显示 |
| `OBSOLETE` | 追踪菜单中不显示 |

---

## Schema 类型引用

- `(VitaminID)` - 维生素ID引用
- `(EffectTypeID)` - 效果ID引用
