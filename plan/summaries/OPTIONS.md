# OPTIONS.md 总结

## 文档概述

游戏选项定义了玩家可配置的各种设置。**无新大类定义**，为配置系统文档。

---

## 选项类型

| 类型 | 说明 |
|------|------|
| `string_input` | 字符串输入 |
| `int_input` | 整数输入 |
| `float_input` | 浮点数输入 |
| `toggle` | 开关选项 |
| `selection` | 选择选项 |

---

## 通用字段

| 字段 | 说明 |
|------|------|
| `name` | 选项名称 |
| `description` | 选项描述 |
| `default` | 默认值 |
| `info` | 帮助信息 |

---

## selection 字段

| 字段 | 说明 |
|------|------|
| `items` | 可选项数组 |
| `default` | 默认选项索引 |
