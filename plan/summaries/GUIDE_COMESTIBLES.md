# GUIDE_COMESTIBLES.md 总结

## 文档概述

消耗品指南，说明如何分类和组织消耗品JSON文件。**无新大类定义**，仅为文件组织指南。

---

## 文件分类

### 特殊消耗品

| 文件 | 说明 |
|------|------|
| `med.json` | 有成瘾效果的消耗品(除酒精和咖啡因)，绷带和消毒剂类物品 |
| `mre.json` | MRE相关物品和消耗品 |
| `mutagen.json` | 有变异剂效果的消耗品 |
| `carnivore.json` | 从动物或怪物身上获取的物品 |
| `protein.json` | 蛋白质相关消耗品 |
| `spice.json` | 香料 |
| `frozen.json` | 冷冻食品 |
| `brewing.json` | 酿造相关 |

### 液体消耗品

| 文件 | 说明 |
|------|------|
| `alcohol.json` | 有酒精成瘾的饮料 |
| `soup.json` | 汤类消耗品 |
| `drink.json` | 饮料(茶、果汁、水等) |
| `drink_other.json` | 其他饮料(醋、芥末等) |

### 固体消耗品

| 文件 | 说明 |
|------|------|
| `junkfood.json` | 垃圾食品 |
| `sandwich.json` | 三明治 |
| `meat_dishes.json` | 肉类菜肴 |
| `raw_veggy.json` | 生蔬菜 |
| `raw_fruit.json` | 生水果 |
| `veggy_dishes.json` | 素菜 |
| `bread.json` | 面包 |
| `wheat.json` | 小麦相关 |
| `egg.json` | 蛋类 |
| `dairy.json` | 乳制品 |
| `mushroom.json` | 蘑菇 |
